- 1d6 = main nue
- 2d6 = arme/outil improvisée
- 3d6 = (vs Cuire) arme petite (dague) / outil approprié 
- 4d6 = hachette, épée courte / 
- 5d6 = (vs Maille) arme longue, épée longue / outil spécialisé 
- 6d6 = arme batarde {deux mains ou une}
- 7d6 = (vs Plate) arme lourde / halbarde, hache d’arme {lent}
