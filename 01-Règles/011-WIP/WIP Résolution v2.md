# Tests d'Action
tous les tests proactif des PJ fonctionne comme dans [[Résolutions]]
### Mécanique de base

**D20 roll-under Attribut pertinent** (VIG / FIN / ESP selon la nature de l'action).

Le résultat est lu sur le **spectre de succès gradué** :

| Résultat sur le dé                | Tier               | Effet sur test instantané        | Effet sur tes de progresion |
| --------------------------------- | ------------------ | -------------------------------- | --------------------------- |
| 16–20 sous                        | **T4 — Total**     | -1 Tenssion                      | dé de progression [+4d]     |
| 11–15 sous                        | **T3 — Critique**  | -1 Tenssion                      | dé de progression [+3d]     |
| 6–10 sous                         | **T2 — Marqué**    | -1 Tenssion                      | dé de progression [+2d]     |
| 0–5 sous l'attribut               | **T1 — Normal**    | Résultat escompté                | dé de progression [+1d]     |
| Égal à l'attribut, ou ≤ blessures | **T0 — Impasse**   | Payer 1d6 END pour T1, sinon T-1 | Lancer dé de progression    |
| 16–20 au-dessus                   | **T-1 — Normal**   | Rien de se passe                 | dé de progression [-1d]     |
| 11–15 au-dessus<br>               | **T-2 — Marqué**   | +1 Tenssion                      | dé de progression [-2d]     |
| 6–10 au-dessus                    | **T-3 — Critique** | +1 Tenssion                      | dé de progression [-3d]     |
| 1–5 au-dessus                     | **T-4 — Total**    | +1 Tenssion                      | dé de progression [-4d]     |
> Il y avait une erreur dans la base

### Action instantanée
D'autres effet peuvent être ajouter ou substitué selon le contexte, au choix du MJ.
### Action progressive

> [!NOTE]  Exemple
> Test FIN. La FIN est des 13. Lancer D20, résultat 12 -> Succès Critique Tier 3. Lancer 1d6 de progression [+3d].

#### Tâches 
Le MJ détermine l'**Ampleur** représenté par des tokens et le **Seuil** de complexité 
Lancer le ou les dés de progression, d6 par défaut ou dé d'outil et rajouter [+/-Td] au jet
